## 🧠 Featured AI Product

### 🤖 IIT PMRecruit AI — Intelligent Recruitment Platform

🔗 GitHub: https://github.com/MikeGSA/IIT-PMRecruit-AI  

An **AI-powered recruitment screening platform** combining a modern frontend with an **agent-based backend (n8n + LLMs)** to automate candidate evaluation.

### 🚀 Why This Matters (Product Thinking)

- Solves **high-friction hiring workflows**
- Reduces manual screening effort using **AI agents**
- Improves hiring efficiency with **data-driven scoring**
- Demonstrates **Agentic AI orchestration in real-world use**

### 🧩 Key Capabilities

- 📄 Resume parsing & candidate profiling  
- 🧠 AI-driven candidate scoring system  
- 📅 Automated interview scheduling  
- 📊 Real-time evaluation dashboard  
- ✉️ Automated rejection & communication workflows

- **Agent 1** → Resume Parsing  
- **Agent 2** → Candidate Scoring  
- **Agent 3** → Interview Scheduling  
- **Agent 2b** → Rejection Automation
